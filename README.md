# Joshi Hospital Website

A modern, responsive website for Joshi Hospital in Shirdi, showcasing quality healthcare services since 1982.

## Features

- **Modern Design**: Clean, professional healthcare-themed design
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Elements**: JavaScript-powered interactivity including:
  - Welcome alert messages
  - Automatic footer year update
  - Smooth scrolling navigation
  - Active navigation highlighting
  - Contact form with validation
  - Scroll-triggered animations

## Sections

- **Home**: Hero section with compelling healthcare messaging
- **About**: Hospital history and key statistics
- **Services**: Medical services offered (General Medicine, Emergency Care, Diagnostics, Surgery)
- **Doctors**: Medical team information
- **Contact**: Contact information and contact form

## JavaScript Functionality

The website includes several interactive features:

1. **Welcome Alert**: `showAlert()` function displays a welcome message
2. **Dynamic Footer**: Automatically updates copyright year
3. **Smooth Scrolling**: Navigation links smoothly scroll to sections
4. **Active Navigation**: Highlights current section in navigation
5. **Form Validation**: Contact form with client-side validation
6. **Scroll Animations**: Cards animate into view as user scrolls

## Technologies Used

- HTML5
- CSS3 (with modern features like Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Inter font family)

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. The website is ready to use - no build process required
